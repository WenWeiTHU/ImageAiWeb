About us

This is a simple website developed with Python3 that provides users with deep learning-based image processing services.

The website uses django architecture and its image recognition service is based on imageai. For user-uploaded images or URIs, this site uses the hottest ResNet and SqueezeNet for image classification (ResNet50 by Microsoft Research - model size: 98 MB, with faster prediction time, higher accuracy; SqueezeNet - model size: 4.82 MB, with the shortest prediction time and moderate accuracy). And for object detection ,we use the most advanced RetinaNet, we also plan to add more models' supports and algorithms in subsequent versions.

When you use this site, please enter the registration interface first. After logon successfully, the site will automatically jump to the login interface. After entering the homepage, you can choose to upload an image or a valid web page image link, after which the image will be processed in the backend. Then finally it will return the results consists of three parts: the type prediction of the image by two models(including the category and corresponding probability, the default returns the five largest types of possibilities), and the output of the image after the objects are detected (labeled on the basis of the original image with objects detected and its confidence level). When you'd like to quit safety, please click the logout button.

On the home page, you can also query your history records or select specific records filter by time period, or simply delete selected ones.
